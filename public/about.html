FilmSPA

FilmSPA is an application built for GA for instructor evaluation. It is a basic single page app that persists data through a Sinatra backend.

Films are searched via OMDB directly from the front end. Results are returned, parsed, and rendered into a basic table.

Clicking individual films generate a further ajax request which populates a standard modal for displaying the additional information.

Clicking the favorites button simply sends the information to POST route for favorites with the name and id. The backend persists the data in a JSON flat file.
